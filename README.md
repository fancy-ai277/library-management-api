# Library Management API

## Project Description
This is a Django-based Library Management API designed to manage books and borrowing records in a library system.  
It provides backend functionality for adding, viewing, and managing library resources through the Django admin interface.

---

## Features
- Manage books (add, update, delete)
- Track borrowing records
- Admin dashboard for managing data
- REST API-ready backend using Django REST Framework

---

## Technologies Used
- Python
- Django
- Django REST Framework
- SQLite (default database)
